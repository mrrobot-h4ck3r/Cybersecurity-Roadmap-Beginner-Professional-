# 📖 2. History of Linux

Before learning Linux, it's helpful to understand where it came from and why it became one of the most popular operating systems in the world.

---

# 🌍 The Story Begins

In the early days of computers, most operating systems were expensive and closed-source. Only the company that created them could modify or improve them.

One of the most popular operating systems at that time was **UNIX**.

UNIX was powerful, stable, and mainly used by universities, research organizations, and large companies. However, it was expensive and not freely available to everyone.

People wanted a free operating system that anyone could use and improve.

---

# 🐧 The GNU Project

In 1983, **Richard Stallman** started the **GNU Project**.

The goal of the GNU Project was simple:

> Create a completely free operating system.

The GNU Project created many important tools such as:

- GCC (Compiler)
- Bash Shell
- Core Utilities
- Text Editors

However, one important piece was still missing...

The **Kernel**.

---

# 🧠 What is a Kernel?

The **Kernel** is the heart of an operating system.

It manages:

- CPU
- RAM
- Storage
- Devices
- Applications

Think of the kernel as the **brain** of the operating system.

Without a kernel, an operating system cannot function.

---

# 👨‍💻 Linus Torvalds

In 1991, a Finnish computer science student named **Linus Torvalds** wanted a free operating system for his personal computer.

Instead of waiting for someone else to build it, he started writing his own kernel.

This project became **Linux**.

He shared the source code with the world.

Developers from around the globe started improving it.

Linux quickly became one of the largest open-source projects ever created.

---

# 🤝 GNU + Linux

The GNU Project already had many tools.

Linux provided the missing kernel.

Together they formed what many people call **GNU/Linux**.

When you install Ubuntu, Debian, or Kali Linux, you are actually using:

GNU Tools + Linux Kernel

---

# 🌎 Linux Today

Today Linux is everywhere.

It powers:

- Web Servers
- Android Phones
- Cloud Platforms
- Data Centers
- Supercomputers
- IoT Devices
- Smart TVs
- Routers
- Security Tools

Most websites you visit every day are hosted on Linux servers.

---

# 🎯 Why Does This Matter?

Understanding Linux history helps you understand why Linux is:

- Free
- Open Source

- Highly Secure
- Constantly Improving

Thousands of developers worldwide contribute to Linux every day.

# 🐧 Chapter 2 - Linux Fundamentals

> "Before becoming a Cybersecurity Professional, learn the operating system that powers most of the Internet."

---

# 📖 Introduction

Welcome to the Linux chapter!

Linux is one of the most important skills in Cybersecurity. Whether you want to become a SOC Analyst, Penetration Tester, Bug Bounty Hunter, Cloud Engineer, or Security Engineer, learning Linux is essential.

Don't worry if you've never used Linux before. This chapter starts from **absolute zero** and gradually moves to advanced topics.

By the end of this chapter, you'll understand how Linux works, how to use the terminal, manage files and users, troubleshoot systems, and perform tasks commonly used in cybersecurity.

---

# 🎯 Learning Objectives

After completing this chapter, you will be able to:

- Understand what Linux is.
- Know the history of Linux.
- Explain why Linux is widely used.
- Identify different Linux distributions.
- Navigate the Linux file system.
- Use the Linux terminal confidently.
- Execute basic to advanced Linux commands.
- Manage users and permissions.
- Understand Linux networking basics.
- Write basic Bash scripts.
- Use Linux in cybersecurity.

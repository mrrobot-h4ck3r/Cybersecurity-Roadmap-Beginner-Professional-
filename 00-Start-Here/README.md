# 🧠 How To Take Notes

Don't copy everything.

Write notes in your own words.

For every topic answer these questions

What is it?

Why do we use it?

How does it work?

Where is it used?

What are the advantages?

What are the disadvantages?

What tools are used?

How is it related to Cybersecurity?

These questions will make your understanding much stronger.

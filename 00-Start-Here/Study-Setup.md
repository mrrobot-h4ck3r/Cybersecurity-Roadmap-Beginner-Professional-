# Study Setup

Recommended software:

- Google Chrome or Firefox
- Visual Studio Code
- Git
- GitHub Desktop (optional)
- VirtualBox
- VMware Workstation Player (optional)
- 7-Zip
- Notepad++
- Obsidian or Joplin (for notes)

Create these folders on your computer:

Cybersecurity/
│
├── Notes/
├── Labs/
├── Projects/
├── Writeups/
├── Tools/
├── Downloads/
└── Screenshots/

Keeping your work organized from day one will save you a lot of time later.

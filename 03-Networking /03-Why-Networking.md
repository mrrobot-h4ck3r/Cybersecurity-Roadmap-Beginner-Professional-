# 📖 3. Why Networking?

> "Networking is the backbone of the Internet and Cybersecurity."

---

# 🎯 Learning Objectives

After completing this topic, you will:

- Understand why networking is important.
- Learn where networking is used.
- Know why networking is essential for cybersecurity.

---

# 🤔 Why Do We Need Networking?

Networking allows devices to communicate and share information with each other.

Without networking, computers would work independently and would not be able to:

- Access the Internet
- Share files
- Send emails
- Watch videos online
- Play online games
- Use cloud services

---

# 🌍 Where is Networking Used?

You use networking every day, often without realizing it.

Examples:

- 🌐 Browsing websites
- 📧 Sending emails
- 📱 Using WhatsApp
- 🎥 Watching YouTube
- 🎮 Playing online games
- 🛒 Online shopping
- 🏦 Internet banking
- ☁️ Cloud storage (Google Drive, OneDrive)

---

# 🔐 Why is Networking Important in Cybersecurity?

Almost every cyber attack happens over a network.

Examples:

- Phishing attacks
- Malware
- Ransomware
- Website attacks
- Data theft
- Password attacks

If you understand networking, you'll better understand **how attacks happen** and **how to defend against them**.

---

# 💼 Why Should a SOC Analyst Learn Networking?

A SOC Analyst uses networking to:

- Monitor network traffic
- Detect suspicious connections
- Investigate security incidents
- Analyze logs
- Respond to attacks

---

# 🛡 Why Should a Penetration Tester Learn Networking?

A Penetration Tester uses networking to:

- Discover live hosts
- Scan open ports
- Find vulnerabilities
- Understand network architecture
- Test network security

---

# 🐞 Why Should a Bug Bounty Hunter Learn Networking?

A Bug Bounty Hunter uses networking to:

- Understand how websites communicate
- Analyze requests and responses
- Identify exposed services
- Test APIs
- Discover security weaknesses

---

# 🌟 Real-Life Example

When you open **https://google.com**:

1. Your device sends a request.
2. The request travels through the Internet.
3. Google's server receives it.
4. The server sends a response.
5. The webpage loads on your screen.

This entire process happens because of networking.

# 🌐 Chapter 3 - Networking Fundamentals

> "Before securing a network, you must first understand how it works."

---

# 📖 Introduction

Networking is one of the most important topics in Cybersecurity.

Whether you want to become a:

- SOC Analyst
- Penetration Tester
- Bug Bounty Hunter
- Cloud Engineer
- Security Engineer
- Incident Responder

you must understand how computers communicate with each other.

This chapter starts from the absolute basics and gradually moves to advanced networking concepts.

No previous networking knowledge is required.

---

# 🎯 Learning Objectives

After completing this chapter, you will be able to:

- Understand how computer networks work.
- Explain how devices communicate.
- Understand IP Addresses and MAC Addresses.
- Understand the OSI Model and TCP/IP Model.
- Learn common networking protocols.
- Understand DNS, DHCP, and NAT.
- Use basic networking commands.
- Analyze network traffic using Wireshark.
- Perform basic network scanning with Nmap.
- Build a strong networking foundation for cybersecurity.
